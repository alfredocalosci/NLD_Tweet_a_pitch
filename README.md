# Tweet_a_pitch_prj
converts tweets to tone pad music dots

see a demo at:
https://vimeo.com/141057929

1) connect the app to a midi device
2) it will serch for: #education
3) modify the hashtag you want to search for editing: data > "search_for.txt"
